Introduction

JavaScript is a versatile scripting language used for creating dynamic web content.

It can be run in the browser or on the server using environments like Node.js.

console.log("Hello, world!");
  • Client-side language
  • Interpreted language
Variables

Variables store data values using keywords like let, var, or const.

Use `let` for block-scoped variables.

let name = "Harshit";
  • var (function-scoped)
  • let (block-scoped)
Functions

Functions help structure your code into reusable blocks.

They can accept parameters and return values.

function greet() { return "Hello"; }
  • Function Declaration
  • Function Expression
Loops

Loops allow you to run code multiple times.

JavaScript supports for, while, and do-while loops.

for(let i=0; i<5; i++){ console.log(i); }
  • for loop
  • while loop
Arrays

Arrays store multiple values in a single variable.

They use zero-based indexing.

const fruits = ["Apple", "Banana"];
  • Push
  • Pop
